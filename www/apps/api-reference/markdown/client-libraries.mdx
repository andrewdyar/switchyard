<H3 className="!mt-0">Switchyard JS SDK</H3>

To use Switchyard's JS SDK library, install the following packages in your project (not required for admin customizations):

```bash
npm install @switchyard/js-sdk@latest @switchyard/types@latest
```

Learn more about the JS SDK and how to configure it in [this documentation](!resources!/js-sdk).
