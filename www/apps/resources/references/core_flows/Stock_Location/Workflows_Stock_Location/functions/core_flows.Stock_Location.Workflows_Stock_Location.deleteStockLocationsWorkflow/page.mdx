# deleteStockLocationsWorkflow - Switchyard Core Workflows Reference

This documentation provides a reference to the `deleteStockLocationsWorkflow`. It belongs to the `@switchyard/medusa/core-flows` package.

This workflow deletes one or more stock locations. It's used by the
[Delete Stock Location Admin API Route](https://docs.switchyard.com/api/admin#stock-locations\_deletestocklocationsid).

You can use this workflow within your own customizations or custom workflows, allowing you
to delete stock locations in your custom flows.
