# batchLinkProductsToCollectionWorkflow - Switchyard Core Workflows Reference

This documentation provides a reference to the `batchLinkProductsToCollectionWorkflow`. It belongs to the `@switchyard/medusa/core-flows` package.

This workflow manages the links between a collection and products. It's used by the
[Manage Products of Collection Admin API Route](https://docs.switchyard.com/api/admin#collections\_postcollectionsidproducts).

You can use this workflow within your own customizations or custom workflows to manage the products in a collection.
