# cancelPaymentStep - Switchyard Core Workflows Reference

This documentation provides a reference to the `cancelPaymentStep`. It belongs to the `@switchyard/medusa/core-flows` package.

This step cancels one or more payments.

```ts title="src/workflows/my-workflow.ts"
import { createWorkflow } from "@switchyard/framework/workflows-sdk"
import { cancelPaymentStep } from "@switchyard/medusa/core-flows"

const myWorkflow = createWorkflow(
  "my-workflow",
  () => {
    const data = cancelPaymentStep({
      "paymentIds": "{value}"
    })
  }
)
```
