# deleteApiKeysWorkflow - Switchyard Core Workflows Reference

This documentation provides a reference to the `deleteApiKeysWorkflow`. It belongs to the `@switchyard/medusa/core-flows` package.

This workflow deletes one or more secret or publishable API keys. It's used by the
[Delete API Key Admin API Route](https://docs.switchyard.com/api/admin#api-keys\_deleteapikeysid).

You can use this workflow within your customizations or your own custom workflows, allowing you to
delete API keys within your custom flows.
