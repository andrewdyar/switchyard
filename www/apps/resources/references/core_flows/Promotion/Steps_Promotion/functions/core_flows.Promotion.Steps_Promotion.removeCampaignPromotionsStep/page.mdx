# removeCampaignPromotionsStep - Switchyard Core Workflows Reference

This documentation provides a reference to the `removeCampaignPromotionsStep`. It belongs to the `@switchyard/medusa/core-flows` package.

This step removes promotions from a campaigns.

## Example

```ts
const data = removeCampaignPromotionsStep([
  {
    id: "camp_123",
    remove: ["promo_321"]

])
```
