# deletePriceListsWorkflow - Switchyard Core Workflows Reference

This documentation provides a reference to the `deletePriceListsWorkflow`. It belongs to the `@switchyard/medusa/core-flows` package.

This workflow deletes one or more price lists. It's used by the
[Delete Price List Admin API Route](https://docs.switchyard.com/api/admin#price-lists\_deletepricelistsid).

You can use this workflow within your customizations or your own custom workflows, allowing you to
delete price lists in your custom flows.
