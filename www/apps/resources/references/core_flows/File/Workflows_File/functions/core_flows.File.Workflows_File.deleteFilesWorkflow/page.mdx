# deleteFilesWorkflow - Switchyard Core Workflows Reference

This documentation provides a reference to the `deleteFilesWorkflow`. It belongs to the `@switchyard/medusa/core-flows` package.

This workflow deletes one or more files. It's used by the
[Delete File Upload Admin API Route](https://docs.switchyard.com/api/admin#uploads\_deleteuploadsid).

The [File Module Provider](https://docs.switchyard.com/resources/infrastructure-modules/file) installed
in your application will be used to delete the file from storage.

You can use this workflow within your customizations or your own custom workflows, allowing you to
delete files within your custom flows.

} from "@switchyard/framework/http"

  req: SwitchyardRequest,
  res: SwitchyardResponse

      }
